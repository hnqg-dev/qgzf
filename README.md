# 全哥账房
===============

> 推荐NAS私有云最低配置:CPU:4核、内存：8G、硬盘：40G<br>
> NAS私有云要能支持并安装虚拟机，因各品牌的虚拟机软件不同，不做推荐；<br>
> 虚拟机安装系统推荐：CentOS7.9或Ubuntu22.0；<br>
> 安装运行环境：[宝塔面板](https://www.bt.cn)；<br>
> 虚拟机安装系统教程可以自行查找.<br>

## 当前功能

* 第一次运行apk,可以绑定自己的全哥账房系统Url，记得配置端口;
* 支持多用户注册
* 支持多资产账户管理
* 支持多账本管理
* 账本的分类独立管理
